# mini-game-jamsoft
Um mini-game educativo desenvolvido como projeto de teste, focado no aprendizado interativo sobre Inteligência Artificial e o conceito de harness. No jogo, o jogador explora a empresa Jamsoft, coleta passes e aprende na prática os principais pontos do uso e funcionamento de IA, com o objetivo de ser “contratado” ao dominar todos os conceitos.
